RESUME FILE INSTRUCTIONS
========================

To add your resume to the portfolio:

1. Save your resume as a PDF file
2. Rename it to: resume.pdf
3. Place it in this folder: /Users/prashanth/my-portfolio/documents/

The download link will automatically work once the file is in place.

If you want to use a different filename:
- Update the href in index.html (line 97)
- Update the href in js/main.js (line 201)
